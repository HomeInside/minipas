# hola mundo

```mp
program helloworld;
begin
    writeln("Hello World in minipas!");
end.
```

Guarda este fragmento de código, en un archivo llamado **hello_world.mp**, en lugar conveniente.

Ahora haz lo siguiente:

```bash
$ minipas run hello_world.mp
```
la salida será algo como:
```bash
minipas v.1.9.0
Hello World in minipas!
```

y eso es todo!

para conocer más acerca de como crear programas en **Minipas**, consulta [Estructura de un programa.](../lang/program_structure.md)
